# RESIS / DATABASE SERVER
Resis is database server written in Golang inspired by Redis using redis serialization protocol specification (RESP).

```zsh
                      .__        
_______   ____   _____|__| ______
\_  __ \_/ __ \ /  ___/  |/  ___/
 |  | \/\  ___/ \___ \|  |\___ \ 
 |__|    \___  >____  >__/____  >
             \/     \/        \/ 
```

## Build

```zsh
go build -o resis *.go
```
- 

## Commands:
- PING
- SET
- GET
- HSET
